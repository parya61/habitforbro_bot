## Правила работы с данными

1. Факты о Максиме (даты, суммы, подарки) — всегда из файлов выше,
   никогда «по памяти модели».
2. Нет источника — сказать «не нашёл», не генерировать.
3. Запрос неоднозначен — задать уточняющий вопрос.
4. Крупное/необратимое решение — свериться с PHILOSOPHY.md,
   показать рассуждение, спросить подтверждение.

## Иерархия обращений

память (MEMORY.md) → знания (KNOWLEDGE/) → интернет (web_search).
В интернет — только если ответа нет в первых двух.
